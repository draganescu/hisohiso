hisohiso + Hermes

Hermes,
in a room of its own.

Hermes is an autonomous agent that can use tools. Hisohiso gives it a private local channel: Hermes runs on your machine, and the phone gets touch-friendly decisions, output, and follow-up without turning the work into a Slack thread, chat bot, or full coding-agent cockpit.

Why not just use Slack or Telegram?

Those tools are built around people and workspaces. Hermes needs a cleaner line: one local agent session, one encrypted channel, and output that can become buttons, diffs, command logs, carousels, and confirmations instead of a noisy message thread.

This matters most when Hermes is connected to private tools, local files, or local models with sensitive data. With hisohiso, it is you and your agent; not you, your agent, and a chat company in the middle.

Install the Hermes setup skill.

The skill teaches Hermes how to set up the wrapper and register it with the hisohiso daemon. If you are self-taught or just experimenting, this is the easier path: ask Hermes to do the setup, then read what it changed.

hermes skills install https://hisohiso.org/skills/hermes.md

Then run Hermes and ask it to use the hisohiso Hermes bridge skill to finish setup.

What the setup creates.

  • A hisohiso CLI install. The setup checks that the local hisohiso command works.
  • A Hermes wrapper. The wrapper keeps one Hermes session id per encrypted channel, so sessions do not blur together.
  • A daemon agent named hermes. Send hermes in the control channel to spawn a new Hermes-backed channel.
  • Phone-friendly replies. Hermes can send diffs, terminal output, file trees, buttons, and confirm dialogs as real hisohiso UI blocks.

What it feels like in your hand.

You start the daemon once, pair your phone, and keep the control channel around. Later, when you want help, send hermes. A new channel appears for that session, separate from your personal chats and workspaces.

hisohiso daemon start
# from the control channel on your phone:
hermes

When Hermes proposes a command or shows a diff, the phone can render it as a native card instead of a wall of chat text.

A few honest limits.

This is for people who are comfortable running local tools. Hisohiso does not install Hermes for you, manage your API keys, or make unsafe commands safe. It gives you a private, self-hostable communication channel instead of a general messaging app. You still decide what the agent is allowed to do.

Hermes does the work. Hisohiso keeps the line clean.
